Installation
============

Download from `GitHub <https://github.com/s2n-cnit/scms>`.

.. code-block:: console

    git clone https://github.com/s2n-cnit/scms.git
    cd scms
    pip install -r requirements.txt


`SCMS` requires `Python` version 3.5+.
It's also working with `PyPy` and `PyPy3`.
